## How to use 🤷‍♂️

---

1. [https://gijol.im](https://gijol.im) 에 접속한다.
2. zeus에서 성적표를 다운로드 한다. (웹 내에 설명되어 있음)
3. gijol에 성적표를 업로드하고, 결과를 확인한다.
